this is a responsive web page open in desktop site to view its effect from android / resize browser screen to see effect

css style position:static postion the element to default unaffected by top left bottom right etc
position:relative set the position relative to normal position (here left:30px; width:50px)
position:absolute sets the position relative to parent /if no parent then relative to body , works only if top , left etc are provided
position:fixed sets the fixed position to a viewport unaffected by scrolls
position:sticky sets element position relative initially then fix the position (unaffected by scroll)
